## Rookie component library

React组件库

### 安装最后已经发布的组件库来试试

~~~javascript
npm install rookie-design --save
~~~

### 使用

~~~javascript
// 加载样式
import 'rookie-design/dist/index.css'
// 引入组件
import { Button } from 'rookie-design'
~~~



### 一些本地开发命令

~~~bash
//启动本地环境
npm run stroybook

//跑单元测试
npm test

//build可发布静态文件
npm run build

//发布到 npm
npm run publish
~~~
